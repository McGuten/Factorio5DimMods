## Overview

5Dim's Decoration adds decorative elements to personalize and organize your factory. Make your base beautiful and easy to navigate!

## Features

### 🔢 Placeable Numbers (0-9)

Large, visible numbers you can place on the ground:

| Item | Use Case |
|------|----------|
| **0-9** | Label train stations, production lines |

Perfect for:
- Train station numbering
- Production line identification
- Base section markers

### 🔤 Placeable Letters (A-Z)

Full alphabet for labeling:

| Item | Use Case |
|------|----------|
| **A-Z** | Section names, category labels |

Perfect for:
- "IRON", "COPPER", "STEEL" section labels
- Production area naming
- Base organization

### 🎨 Decorative Tiles

Visual floor tiles for organization:

| Type | Use Case |
|------|----------|
| **Colored tiles** | Zone separation |
| **Pattern tiles** | Walkways, paths |
| **Warning tiles** | Danger zones |

### 🪧 Signs and Markers

Additional organizational tools:

| Type | Use Case |
|------|----------|
| **Directional signs** | Traffic flow |
| **Warning signs** | Hazard areas |
| **Info markers** | Points of interest |

## Factory Organization Tips

### Color Coding

| Color | Suggested Use |
|-------|--------------|
| 🔴 Red | Copper processing |
| 🔵 Blue | Circuits |
| 🟢 Green | Iron/Steel |
| 🟡 Yellow | Oil/Fluids |
| 🟣 Purple | Science |
| ⚫ Black | Power |

### Numbering Systems

**Train Stations:**
```
1 - Iron Ore Loading
2 - Copper Ore Loading
3 - Steel Unloading
...
```

**Production Lines:**
```
A - Green Circuits
B - Red Circuits
C - Blue Circuits
...
```

## Dependencies

- **Required**: `5dim_core >= 2.0.42`

## Installation

1. Install `5dim_core` first
2. Download from [Factorio Mod Portal](https://mods.factorio.com/user/McGuten)
3. Enable in the mod menu
