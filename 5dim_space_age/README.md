5Dim's Space Age expands all Space Age DLC machines with 10 tiers each. Conquer the planets with upgraded equipment!

## Planet Strategy

### Vulcanus
- Upgrade foundries first for faster metallurgy
- Big mining drills essential for lava resources

### Gleba
- Biochambers for bioflux production
- Agricultural towers for sustainable farming

### Fulgora
- EM plants for scrap recycling
- Recyclers for material recovery

### Aquilo
- Heating towers to survive the cold
- Cryogenic plants for fusion materials

### Space Platform
- Upgrade cargo bays for longer trips
- Better collectors = more asteroid resources

## Dependencies

- **Required**: `5dim_core >= 2.0.42`
- **Required**: `space-age >= 2.0.0`
- **Required**: `quality >= 2.0.0`

## Installation

1. Own and install Space Age DLC
2. Install `5dim_core` first
3. Download from [Factorio Mod Portal](https://mods.factorio.com/user/McGuten)
4. Enable in the mod menu
